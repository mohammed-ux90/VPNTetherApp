# تطيبق VPN USB Tethering

هذا المشروع يوفر كود مصدري (Source Code) جاهز لتطبيق أندرويد يساعدك على مشاركة اتصال الـ VPN الخاصة بهاتفك مع الكمبيوتر الموصل به عبر طريق الـ USB Tethering أو نقطة اتصال الهاتف (Wi-Fi Hotspot) وذلك **بدون الحاجة لصلاحيات الروت (Root)**.

## كيف يعمل؟
نظام أندرويد لا يسمح بتمرير حركة البيانات الخاصة بـ Tethering عبر الـ VPN بشكل افتراضي لحماية البيانات.
هذا التطبيق يقوم بحل المشكلة عن طريق إنشاء **Local HTTP Proxy Server** مباشر داخل هاتفك يعمل من خلال الـ VPN.
عندما يقوم الكمبيوتر الخاص بك بالاتصال بالانترنت عبر الهاتف، يمكنك فقط إدخال إعدادات الـ Proxy في الكمبيوتر ليمر الاتصال تلقائياً عبر النفق السري (VPN Tunnel).

## خطوات الاستخدام (للمطور)
1. قم بفتح برنامج **Android Studio**.
2. اختر **Open an existing Android Studio project**.
3. قم بتحديد المجلد `VPNTetherApp`.
4. انتظر قليلاً حتى يعمل Gradle Sync ويقوم بتحميل المكتبات (تم استخدام `LittleProxy`).
5. قم بعمل Build واستخراج الـ APK لنشره أو تثبيته مباشرة على جوالك.

## خطوات الاستخدام (للمستخدم النهائي)
1. **في الهاتف:** قم بتشغيل اتصال الـ VPN المفضل لديك (مثلاً: ExpressVPN, NordVPN).
2. قم بتفعيل **USB Tethering** (تقييد USB) من إعدادات الهاتف بعد توصيل الكيبل بالكمبيوتر.
3. افتح هذا التطبيق واضغط على **Start VPN Tethering**.
4. سيظهر لك الآيبي (مثلاً: `192.168.42.129`) ومنفذ (`8080`).
5. **في الكمبيوتر:**
   - اذهب إلى إعدادات الشبكة (Network & Internet) -> إعدادات الـ Proxy.
   - قم بتفعيل الـ Manual Proxy Setup.
   - اكتب رقم الـ IP والـ Port الذي ظهر في التطبيق.
6. انتهى! الآن جميع اتصالات الكمبيوتر مشفرة وتمر عبر الـ VPN الخاص بالهاتف.
